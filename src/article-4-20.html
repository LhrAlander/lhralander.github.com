前言

哈哈哈哈，好久没写东西了啊喂！原因有很多，包括要写密码学的作业但是又懒得写就拖了很久，然后不知道要学什么又耽搁了一段时间，中间陆陆续续看了CSS.Secrets，后面的内容都比较普通所以没做笔记。非要说这几天做了什么的话就是前两天看着自己的github主页这么丑很难受啊喂！但是作为一个准程序员不能忍要用别人的模板！！！！于是就想着手撸一个主页，也是模仿着知乎某答主的主页自己写了一个出来，很丑陋但是写完时成就感满满啊！所以还是要多写代码才知道自己有多菜。这篇记录一下在写主页的时候遇到的问题和自己的理解。

1.对div模型有了新的理解
这个其实是基本功的问题但是还是在看了别人的代码才恍然大悟。经典的盒子模型体积包含了 边框、padding、内容区 但不是因为有体积才有上述三样东西，而是先有了这三样东西才撑大了div！我想到这点的时候简直被自己感动到了啊喂！比如看下面的代码：

.test-div {
    padding: 15px 20px 0 0;
    border: 1px solid #ddd;
}

并没有显示规定div的长宽但是盒子被撑起来了，在我自己写代码的过程中不止一次边感叹自己太蠢边写下这样的代码。OTLL
而且我认为在很多时候都应该是由内容撑起盒子的高度。（不知道是不是这样的）
2.对于页面的布局往大了去看
记得第一次练习老师就这么说过，我以为我理解了老师的意思但是其实真正做起来还是差的很远啊。页面并不只是一个个标签还要考虑到里面的数据内容，以此方便配合json和ajax使用在布局的第一步就要想好哪些标签里填写哪些数据。因为毫无准备就开始动手写这个主页，只有到了写ajax的时候才懊恼自己的布局实在是烂，但还是臭不要脸地硬着头皮就那样写完了。
在这里主要是提醒自己下次写网页一定要想好哪些容器放什么内容再进行布局！

最后

若说收获一个简单的主页怎么可能有那么多收获。知识熟悉了一下怎么操作json数据，是时候开始着手准备期末的课题了，每天都很忙但都不知道在忙什么也是蛮可怜的。
